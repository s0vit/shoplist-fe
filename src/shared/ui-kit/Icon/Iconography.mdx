# Iconography

Наша библиотека иконок содержит все необходимые иконки для интерфейса приложения.

## Размеры иконок

Доступные размеры иконок:
- **xs**: 12px
- **sm**: 16px  
- **md**: 24px (по умолчанию)
- **lg**: 32px
- **xl**: 48px

## Использование

```tsx
import { Icon } from '@/shared/ui-kit/Icon';

// Базовая иконка
<Icon name="plus" size="md" />

// С кастомным размером
<Icon name="home" size={20} />

// С цветом
<Icon name="plus" size="md" color="#FF0000" />

// Кликабельная иконка
<Icon name="close" size="sm" onClick={handleClose} />
```
